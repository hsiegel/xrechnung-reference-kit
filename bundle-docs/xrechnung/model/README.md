# XRechnung SeMoX Models

[SeMoX models](https://semo-xml.org/en/about/) of the Standard XRechnung (CIUS and Extension).


## CIUS XRechnung

The model of CIUS XRechnung is defined in [xrechnung-model-semox.xml](xrechnung-model-semox.xml). 

## Extension XRechnung

The model of Extension XRechnung ([xrechnung-extension-model.xml](xrechnung-extension-model.xml)) 
is derived from the CIUS XRechnung model by _Tailoring_.

## CIUS XRechnung CVD

The model of CIUS XRechnung CVD ([xrechnung-cvd-model.xml](xrechnung-cvd-model.xml)) 
is derived from the CIUS XRechnung model by _Tailoring_.

## Background on SeMoX

For information on the concept of SeMoX please visit [semo-xml.org](https://semo-xml.org/en/about/). 


In short, SeMoX defines a structured representation of Terms (Business Terms), Datatypes, Rules (Business rules), Structures (i.e. groups and cardinalities), Syntax Mappings, and codelist references in XML.

XSD Schema of the model files are available in [GitLab](https://projekte.kosit.org/semox/semox-model/), the core artifact being [semo.xsd](https://projekte.kosit.org/semox/semox-model/-/blob/master/src/main/xsd/semo.xsd).

Further information on the concept of tailoring can be found [here](https://projekte.kosit.org/semox/semox-model/-/blob/master/src/main/doc/tailoring.md).
